# Library API - REST API with Swagger

This project is a simple ASP.NET Core Web API for managing a library system.  
It is designed for students to learn how to build and test REST APIs using Swagger.

## Project Goal

The purpose of this project is to help students understand:

- how a REST API works
- how to run an ASP.NET Core Web API project
- how to test API endpoints using Swagger UI
- how to organize a simple backend project

## Requirements

Before running this project, make sure you have:

- .NET SDK installed
- Visual Studio Code or Visual Studio
- Git installed
- Internet browser for Swagger UI

## 🚀 How to Run the Project

### Step 1: Clone the repository

```bash
git clone git@github.com:amgh2025/Rest_API_Swagger_2026.git
cd Rest_API_Swagger_2026
Step 2: Go to the project folder
cd LibraryApi
Step 3: Restore dependencies
dotnet restore
Step 4: Run the application
dotnet run
Step 5: Open Swagger UI

After running the project, check the terminal for a URL like:

http://localhost:5000

or

https://localhost:7000

Open your browser and go to:

http://localhost:5000/swagger

or

https://localhost:7000/swagger
